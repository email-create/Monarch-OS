# Monarch OS — JW Monarch Land Wholesaling Dashboard

## Quick Deploy to Vercel

### 1. Push to GitHub
1. Go to github.com → Click "+" → "New repository"
2. Name it `monarch-os` → Click "Create repository"
3. Upload all these project files to that repo

### 2. Deploy on Vercel
1. Go to vercel.com → Sign in with GitHub
2. Click "Add New Project"
3. Select your `monarch-os` repo
4. Click "Deploy" — that's it!
5. Your app will be live at `monarch-os.vercel.app`

### Local Development
```bash
npm install
npm run dev
```

## Features
- 16 operational sections for land wholesaling
- Multi-user profiles (Jorge & Winston)
- Real-time sync via Firebase Firestore
- Shared buyers & market data across profiles
- Personal sellers, calls, deals, feedback per profile
- Compare view for side-by-side profile analysis
